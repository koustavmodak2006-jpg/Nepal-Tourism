# 🇳🇵 Nepal Tourism Website

A modern and responsive tourism website showcasing the beauty, culture, and adventure of Nepal. Built using HTML, CSS, Bootstrap, and JavaScript, this project highlights popular destinations, trekking routes, and travel experiences through an attractive and user-friendly interface.

## ✨ Features

* Responsive and modern UI
* Stunning hero section with Nepal-inspired visuals
* Popular destinations showcase
* Everest Base Camp Trek section
* Travel highlights and services
* Clean card-based layout
* Mobile-friendly design
* Smooth navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## 📍 Featured Destinations

* Kathmandu
* Pokhara
* Chitwan National Park
* Lumbini
* Mustang
* Everest Base Camp Trek
* Bandipur

This project was created to improve my front-end web development skills and demonstrate my ability to build visually appealing and responsive websites.
